# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability, please report it through [GitHub Security Advisories](https://github.com/nicarl/sluice/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

You can expect an acknowledgement within 72 hours and a resolution timeline within one week.
